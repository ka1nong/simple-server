# simple-server
simple系列的基础脚手架  

## 自身目录结构
* bin: 可执行脚步
* conf: 配置文件
* doc: 文档
* logs: 日志
* lib: server核心代码

## 部分模块说明
* lib/utils/innerRequest  
```
    使用http请求作为不同simple系列服务相互通信方式
```
* lib/app-manage
```
    应用模块的抽象
```
* lib/dao/dataUtils
```
    未集成ORM相关内容, 以此作为数据操作的工具, 提供一系列的便捷方法
```
