# 规范与说明

## 命名
> 1. 文件夹采用 "-" 连接单词, 如: simple-server, app-manage
> 2. 文件名采用驼峰式
> 3. 变量名采用驼峰式, 局部function名使用 "_" 打头, 如: function _correctArgs() {...}, function _isValid() {...}
> 4. url, params, query采用驼峰式
> 5. 数据库, 表名, 字段名使用 "_" 连接单词

> 2. 8002: simple-school
> 3. 8003: simple-stock
